A set of Implimentations of sorts using an object oriented design.
    All sorts inherit from the basic sort data structure

Run:

    type Make Run
        - Runs the Bubble Sort, Insertion Sort, and the Quick Sort
    
    type Make BubbleSort
        - Runs the Bubble Sort
    
    type Make InsertionSort
        - Runs the Insertion Sort
    
    Type Make QuickSort
        - Runs the Quick Sort
    
    Type Make Random Sort
        - Runs the Random Sort
        (Be careful this one might take a minute)
